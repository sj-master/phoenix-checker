📋 How It Works

  1. Visit Instagram's data download page and request your data
  2. Select JSON format and choose only Followers and Following data
  3. Download the ZIP file when ready and extract it to your computer
  4. Find the followers and following JSON files in the extracted folder
  5. Important: Rename the files to exactly followers.json and following.json if Instagram has given them different names (like followers_1.json or following_data.json). The checker requires these exact file names to work properly!
  6. Upload both renamed files below to get your instant analysis
